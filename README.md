## Frontend stack

- **UI**: `react`, `material-ui`, `classnames`, `scss`
- **Data model**: `redux rtk`
- **Lang**: `typescript`
- **Routing**: `react-router`
- **Lint**: `eslint`, `prettier`, `stylelint`
- **Deployment**: `netlify`
- **Architecture**: `feature-sliced`
